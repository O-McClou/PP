Radikalisierung – Kaufreihenfolge

  • A) Sonderanweisung auf KI-Karte → befolgen
  • B) Dom-Check im Markt + Wakhan gewinnt → kaufen
  • C) Sonst: Roter Pfeil (Reihe) + Karte B / schwarzer Pfeil (Spalte)

Pfeil oben → OBERE Reihe, Hof LINKS. Pfeil unten → UNTERE Reihe, Hof RECHTS.

Bestechung

Spieler beherrscht Kartenregion → Wakhan zahlt Bestechung (= Anzahl Stämme in Rupien). Kann nicht zahlen → Karte abwerfen, kein Hofeintrag.

Wakhans Spione

Auf Hofkarte mit höchster Priorität, deren Region = Region ausgespielter Karte, auf der Wakhan nicht die meisten Spione hat.

Wakhans Straßen

Grenzen gemäß Regionenprioritäten (Karte B, von links), beginnend bei Region der ausgespielten Karte.

Kampf

Wakhan ≥ 1 Armee + anderer Spieler loyale Einheiten. Zerstörung: Stämme → Armeen → Straßen. Sonst Spione auf Hofkarte (höchste Priorität). Mehrere Ziele → roter Pfeil.

Steuern

Spieler vor Marktkarten. Meiste Rupien zuerst. Gleichstand → roter Pfeil.

Marschieren

Nur Armeen (keine Straßen nötig). Zielregionen mit feindlichen Stämmen. Max. Armeen = Stammanzahl. Keine Bewegung wenn Herrschaft verloren ginge.

Bauen

In laut Karte B am weitesten links vermerkter Region mit Wakhan-Herrschaft. So viel Geld wie möglich ausgeben.

Verrat

Hofkarte mit höchster Priorität + Loyalitäts-Beute + Wakhan-Spion. Beute immer nehmen.

Bonusaktionen

Von links. Jede ungenutzte Karte der bevorzugten Farbe führt erste mögliche Aktion aus. Jede Karte nur einmal pro Zug.
